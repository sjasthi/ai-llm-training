# AI & LLM Usage Patterns – What Are They Good At?

A practical, student-friendly guide to how AI and Large Language Models (LLMs) are used.

---

## 1. Information & Knowledge Patterns
**Goal:** Find, understand, summarize, or combine information.

### Common Uses
- Q&A over documents (ask questions about PDFs, notes, or code)
- Search + summarize
- Knowledge assistants (HR, IT helpdesk)
- Meeting & email summaries
- Research help

### Example Prompts
- "Explain this school rule in simple words."
- "What are the key points from this story?"

---

## 2. Content Generation Patterns
**Goal:** Create new content.

### Common Uses
- Writing blogs or essays
- Code generation
- Marketing content
- Reports and proposals
- Images or slides

### Example Prompts
- "Write a short story about a robot in school."
- "Create a simple email inviting friends to a birthday party."

---

## 3. Transformation Patterns
**Goal:** Change content into a different format.

### Common Uses
- Summarization
- Translation
- Rewriting tone (formal, friendly)
- Formatting (JSON, Markdown)
- Extracting data from text

### Example Prompts
- "Summarize this paragraph in 2 sentences."
- "Translate this sentence to Spanish."

---

## 4. Classification & Labeling Patterns
**Goal:** Decide what something is.

### Common Uses
- Detecting intent
- Sentiment analysis
- Topic classification
- Spam detection
- Ticket routing

### Example Prompts
- "Is this message positive or negative?"
- "What topic does this paragraph belong to?"

---

## 5. Conversational Agent Patterns
**Goal:** Have natural conversations.

### Common Uses
- Chatbots
- Customer support agents
- Internal copilots
- Tutors or coaches
- Sales assistants

### Example Prompts
- "Teach me multiplication step by step."
- "Help me practice a job interview."

---

## 6. Reasoning & Decision Support Patterns
**Goal:** Help people think, plan, and decide.

### Common Uses
- Explaining decisions
- Comparing options
- Root cause analysis
- Scenario planning
- What-if analysis

### Example Prompts
- "Should I save my money or spend it? Explain why."
- "What happens if we skip homework every day?"

---

## 7. Tool-Using / Action-Taking Patterns
**Goal:** Perform actions using tools.

### Common Uses
- Calling APIs
- Running commands
- Querying databases
- Automation tasks

### Example Prompts
- "Get today’s weather and tell me if I need an umbrella."
- "Find the sum of numbers in this list: 5, 10, 15."

---

## 8. Agentic & Workflow Patterns
**Goal:** Handle multi-step tasks.

### Common Uses
- Breaking big tasks into steps
- Planner–Executor workflows
- Multi-agent collaboration
- Background automation

### Example Prompts
- "Plan a school project step by step."
- "Organize a one-day picnic plan for my class."

---

## 9. Coding & Software Engineering Patterns
**Goal:** Help build and maintain software.

### Common Uses
- Code completion
- Refactoring
- Test generation
- Bug explanations
- Code review

### Example Prompts
- "Write a simple program to add two numbers."
- "Explain what this code does in simple words."

---

## 10. Data & Analytics Patterns
**Goal:** Work with structured data.

### Common Uses
- Natural language to SQL
- Data exploration
- Dashboard explanation
- Detecting anomalies
- Feature engineering help

### Example Prompts
- "Show me the total marks from this table."
- "Explain what this chart means."

---

## 11. Compliance, Safety & Governance Patterns
**Goal:** Reduce risk and follow rules.

### Common Uses
- Policy checking
- Detecting personal data
- Redacting sensitive information
- Audit explanations

### Example Prompts
- "Does this text contain private information?"
- "Remove phone numbers from this paragraph."

---

## 12. Personalization Patterns
**Goal:** Customize for each user.

### Common Uses
- Personalized learning
- Recommendations
- Role-based responses
- Remembering user preferences

### Example Prompts
- "Teach me math based on my level."
- "Recommend books for a 12-year-old who likes space."

---

## Simple Mental Model
Most AI apps do one or more of these:

**Read → Think → Retrieve → Transform → Decide → Act → Explain**
